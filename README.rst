* Documentation: https://aioworkers.readthedocs.io.


Features
--------

* Specify abstract class for communication between components
* Configuration subsystem


Development
-----------

Check code:

.. code-block:: shell

    hatch run lint:all


Format code:

.. code-block:: shell

    hatch run lint:fmt


Run tests:

.. code-block:: shell

    hatch run pytest


Run tests with coverage:

.. code-block:: shell

    hatch run cov
